# Get-NLPowerShellConfig

## SYNOPSIS
Retrieves the current configuration for NLPowerShell.

## SYNTAX

```
Get-NLPowerShellConfig
```

## DESCRIPTION
Displays the current settings for NLPowerShell, excluding the API key for security reasons.
